public bool iter_parent (out TreeIter parent, TreeIter child)

Get the parent node of a child

return parent child description
true valid valid Returns parent node
false invalid valid Child has no parent node
false invalid invalid Logic error

Parameters:

parent

An iterator for the parent node. If the child does not have a parent, or a logic error occurs, the parent iterator is set to invalid.

child

The child iterator to obtain the parent iterator for.

Returns:

True when parent represents a valid iterator
