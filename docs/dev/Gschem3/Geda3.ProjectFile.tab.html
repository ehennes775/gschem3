tab

Description:

public override string tab { protected set; get; }

A short name to appear in widgets

To facilitate renaming the file, this property contains the GLib.FileAttribute.STANDARD_EDIT_NAME.

This property is set by the notify signal handler of the file property. Changing the file property will update this value.
